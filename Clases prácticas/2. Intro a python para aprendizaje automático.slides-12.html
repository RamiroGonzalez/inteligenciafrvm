Introducción¶

Python üêç

Lenguaje de software libre, no tipado, interpretado y orientado a objetos. Se presenta como principal elecci√≥n entre los lenguajes utilizados para aplicaciones de inteligencia artificial, particularmente de machine learning, junto con R, Matlab y Octave (dependiendo la comunidad que lo use).

Posee un enorme ecosistema de librer√≠as de machine learning. Esto sumado a su flexibilidad y su simpleza de uso hacen que sea uno de los lenguajes m√°s populares para la computaci√≥n cient√≠fica.

Algunos IDEs: Jupyter (web), Pycharm, Google Colaboratory.

jupyter notebook / jupyter lab¶

Aplicaci√≥n web de c√≥digo abierto que permite crear y compartir documentos (notebooks) mostrando iterativamente el flujo de c√≥digo en lenguajes como python y R, permitiendo incluir gr√°ficos, explicaciones en Markdown y f√≥rmulas en LaTeX.

Resulta una gran opci√≥n para presentaciones o aplicaciones peque√±as de machine learning ya que permite utilizar el c√≥digo de a fragmentos. Cada fragmento puede ser de texto o de c√≥digo (dependiendo la selecci√≥n del combo box superior: si est√° "Markdown" seleccionado tomar√° a la celda como texto - de estar "Code" tomar√° el fragmento como c√≥digo).

¬øC√≥mo instalarlo?

  • Descargar la plataforma para DataScience "Anaconda" desde

    https://docs.anaconda.com/anaconda/install/

  • Seguir las intrucciones de instalaci√≥n de Anaconda seg√∫n el sistema operativo.

¬øC√≥mo usarlo?

Se recomienda crear un entorno virtual. Desde la consola de Windows o Linux, ejecutar

    conda create -n iafrvm python=3.6

Se activa mediante

    conda activate iafrvm

Para instalar un paquete, ejemplo numpy (tras haber activado el entorno virtual)

conda install numpy

o bien, mediante pip,

pip install numpy

Luego, con el entorno virtual activado, ejecutar

    jupyter notebook

y abrir el notebook desde la interfaz web (tambi√©n pueden usar el comando jupyter lab, que tiene vista de archivos)

Prueben ejecutar y mirar el c√≥digo del notebook de esta clase!

Info adicional:

  • Markdown es un conversor de texto plano a HTML. En los notebooks funciona como suerte de versi√≥n "liviana" de LaTex. Se usa en sitios como Slack, github (para los README), reddit.
  • LaTex es un sistema de tipograf√≠a de alta calidad, que permite armar documentos con una sintaxis predefinida. Su sintaxis es mucho m√°s verborr√°gica comparado con las herramientas t√≠picas de ofim√°tica, pero es mucho m√°s flexible. A LaTex en los notebooks se lo utiliza principalmente para f√≥rmulas.

Info adicional sobre LaTex, para aquellos interesados:

  • Para armar documentos completos en LaTex de forma sencilla una buena alternativa es Overleaf (editor LaTex Web colaborativo). Por su parte, la alternativa que recomendamos es descargarlo mediante TexLive (en Ubuntu, a trav√©s de sudo apt-get install texlive-full (pesa 4Gb)) y editar los documentos mediante TexStudio

Para sus explicaciones en texto, jupyter por defecto utiliza Markdown. Para utilizar LaTex en un notebook, debe encerrarse la sentencia en \$ ecuaci√≥n \\$ para ecuaciones en la misma l√≠nea o bien entre \$\\$ ecuaci√≥n \$\\$ para ecuaciones en una nueva l√≠nea.

Ejemplo $$\lim_{h \to 0} \frac{f(a+h)-f(a)}{h}$$
